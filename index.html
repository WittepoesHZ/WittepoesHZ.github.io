Welkom op Scott's website!

Waarom Deze Opleiding?

Hallo! Mijn naam is Scott Mulder. ik ben 19 jaar oud en ben hier op de HZ terecht gekomen via Scalda, waar ik mijn System and Device diploma heb behaald. aangezien ik nog maar zo jong ben, leek het mij een beetje zonde om gelijk al het beroep in te gaan. daarom ben ik verder gaan studeren!

Ik heb specifiek voor deze opleiding gekozen omdat ik graag wat meer wou leren over de codeer kant. bij Scalda heb ik vooral met hardware gewerkt, en bijna geen code geschreven. tot nu toe bevalt het mij enorm.

Wat Wil Ik Leren?

Ik zou het heel cool vinden om meer te leren over coderen en websites maken. Tot nu toe lukt dat wel, maar ik zou graag meer willen leren. Ik ben tot nu toe erg tevreden met FreeCodeCamp. hier merk ik dat ik veel leer en dat het ook makkelijk is om te stoppen op ieder moment, wat voor mij erg fijn is.
